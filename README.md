BackOffice
==========
Introduction
------------
BackOffice du projet NewWorld.

Le BackOffice est réservé seulement aux personnes autorisé faisant partie du groupe NewWorld.

Pourquoi l'utiliser ?
----------------------
Permet de :
- ``créer``
- ``modifier``
- ``supprimer``

un ``rayon``, ansi qu'un ``produit``.

Dépendances
-----------
- QtCreator 5.2
- MySql
 - Base de données ``listedescourses``
